# psy120l_final_proj
How do attachment styles, particularly attachment anxiety and attachment avoidance, influence willingness to commit in romantic relationships, and does this relationship differ depending on whether the relationship was initiated online or in person?
This project investigates the role of attachment processes in romantic commitment. Participants were randomly assigned to read either an online dating vignette or an in-person relationship vignette and then completed measures of attachment insecurity and relationship commitment. The analyses examined whether attachment style and relationship initiation context predicted willingness to commit in romantic relationships.
This report was created using Quarto and R.
The dataset used for this project is not included in this repository. Therefore, the analyses cannot be reproduced without access to the original data file.
